# 플러그인 작성
플러그인을 작성할 때는 반드시 상단에는 주석으로 플러그인에 대한 소개 및 사용법을 작성하고 이후에 해당 플러그인의 lua스크립트를 작성한다.

주석은 다음과 같은 내용을 포함한다.
- 파일 이름
- 플러그인 이름
- 플러그인 github 주소
- 플러그인에 대한 자세한 설명
- 플러그인 활용 예시
- 플러그인 사전 요구사항 (필요시 작성)
- 플러그인 사용법
- 플러그인 단축키 (커스텀 단축키가 있을 경우 별도로 작성)
