## 1. 모드 전환

| 단축키 | 설명 |
|--------|------|
| `i` | `[모드전환]` 커서 앞에서 입력 모드 전환 |
| `I` | `[모드전환]` 커서를 줄의 첫 글자 앞으로 이동, 입력 모드 전환 |
| `a` | `[모드전환]` 커서 뒤에서 입력 모드 전환 (append) |
| `A` | `[모드전환]` 커서를 줄 끝으로 이동, 입력 모드 전환 |
| `o` | `[모드전환]` 아래에 새 줄 삽입 후 입력 모드 전환 |
| `O` | `[모드전환]` 위에 새 줄 삽입 후 입력 모드 전환 |
| `s` | `[모드전환]` 커서 위치 글자 삭제 후 입력 모드 전환 |
| `S` | `[모드전환]` 현재 줄 전체 삭제 후 입력 모드 전환 |
| `v` | `[모드전환]` 비주얼 모드 전환 (글자 단위 선택) |
| `V` | `[모드전환]` 비주얼 라인 모드 전환 (줄 단위 선택) |
| `<C-v>` | `[모드전환]` 비주얼 블록 모드 전환 (사각형 영역 선택) |
| `R` | `[모드전환]` 교체(Replace) 모드 전환, 텍스트 덮어쓰기 |
| `:` | `[모드전환]` 명령 모드(Command-line) 전환 |
| `<Esc>` 또는 `<C-[>` | `[모드전환]` 노멀 모드로 복귀, 모드 전환 취소 |

## 2. 커서 이동

### 기본 이동
| 단축키 | 설명 |
|--------|------|
| `h` / `j` / `k` / `l` | `[커서이동]` 커서 왼쪽 / 아래 / 위 / 오른쪽 이동 |
| `0` | `[커서이동]` 커서를 줄의 맨 앞으로 이동 (들여쓰기 포함) |
| `^` | `[커서이동]` 커서를 줄의 첫 글자로 이동 (들여쓰기 제외) |
| `$` | `[커서이동]` 커서를 줄의 맨 끝으로 이동 |
| `g_` | `[커서이동]` 커서를 줄의 마지막 공백이 아닌 글자로 이동 |

### 단어 단위 이동
| 단축키 | 설명 |
|--------|------|
| `w` | `[커서이동]` 커서를 다음 단어의 시작으로 이동 |
| `W` | `[커서이동]` 커서를 다음 단어(공백 기준)의 시작으로 이동 |
| `e` | `[커서이동]` 커서를 현재/다음 단어의 끝으로 이동 |
| `E` | `[커서이동]` 커서를 현재/다음 단어(공백 기준)의 끝으로 이동 |
| `b` | `[커서이동]` 커서를 이전 단어의 시작으로 이동 |
| `B` | `[커서이동]` 커서를 이전 단어(공백 기준)의 시작으로 이동 |
| `ge` | `[커서이동]` 커서를 이전 단어의 끝으로 이동 |

### 문서 단위 이동
| 단축키 | 설명 |
|--------|------|
| `gg` | `[커서이동]` 커서를 파일 맨 위로 이동 |
| `G` | `[커서이동]` 커서를 파일 맨 아래로 이동 |
| `{숫자}G` | `[커서이동]` 커서를 지정한 줄 번호로 이동 (예: `42G`) |
| `:{숫자}` | `[커서이동]` 커서를 지정한 줄 번호로 이동 (예: `:42`) |
| `%` | 커서를 짝이 되는 괄호로 이동 `()` `[]` `{}` |
| `(` / `)` | `[커서이동]` 커서를 이전 / 다음 문장 시작으로 이동 |
| `{` / `}` | `[커서이동]` 커서를 이전 / 다음 문단 시작으로 이동 |

### 화면 내 이동
| 단축키 | 설명 |
|--------|------|
| `H` | `[커서이동]` 커서를 화면 맨 위로 이동 (High) |
| `M` | `[커서이동]` 커서를 화면 중간으로 이동 (Middle) |
| `L` | `[커서이동]` 커서를 화면 맨 아래로 이동 (Low) |

### 글자 검색 이동
| 단축키 | 설명 |
|--------|------|
| `f{글자}` | `[커서이동]` 커서를 같은 줄 앞쪽의 해당 글자로 이동 (찾기) |
| `F{글자}` | `[커서이동]` 커서를 같은 줄 뒤쪽의 해당 글자로 이동 (역방향 찾기) |
| `t{글자}` | `[커서이동]` 커서를 앞쪽 해당 글자 직전으로 이동 (till) |
| `T{글자}` | `[커서이동]` 커서를 뒤쪽 해당 글자 직후로 이동 (역방향 till) |
| `;` | `[커서이동]` 직전 `f`/`t` 글자 검색 반복 (같은 방향) |
| `,` | `[커서이동]` 직전 `f`/`t` 글자 검색 반복 (반대 방향) |

### 점프 이력
| 단축키 | 설명 |
|--------|------|
| `<C-o>` | `[커서이동]` 커서를 이전 점프 위치로 이동 (뒤로) |
| `<C-i>` | `[커서이동]` 커서를 다음 점프 위치로 이동 (앞으로) |
| `''` | `[커서이동]` 커서를 직전 점프 위치로 이동 |
| `` `` `` | `[커서이동]` 커서를 직전 편집 위치로 이동 |

### 입력 모드(Insert Mode) 커서 이동
| 단축키 | 설명 |
|--------|------|
| `<S-Right>` | `[커서이동]` 커서를 다음 단어로 이동 (Shift + →) |
| `<S-Left>` | `[커서이동]` 커서를 이전 단어로 이동 (Shift + ←) |
| `<C-Right>` | `[커서이동]` 커서를 다음 WORD로 이동 (Ctrl + →, 공백 기준) |
| `<C-Left>` | `[커서이동]` 커서를 이전 WORD로 이동 (Ctrl + ←, 공백 기준) |
| `<Home>` | `[커서이동]` 커서를 줄의 맨 앞으로 이동 |
| `<End>` | `[커서이동]` 커서를 줄의 맨 끝으로 이동 |
| `<C-o>{모션}` | `[커서이동]` 노멀 모드 명령 1회 실행 후 입력 모드 복귀 (커서 이동 등) |

> **팁**: `<C-o>`를 활용하면 노멀 모드의 모든 이동 명령을 입력 모드에서 사용할 수 있습니다.
> 예: `<C-o>w` (다음 단어), `<C-o>b` (이전 단어), `<C-o>$` (줄 끝), `<C-o>gg` (파일 맨 위)

## 3. 화면 스크롤

| 단축키 | 설명 |
|--------|------|
| `<C-d>` | `[스크롤]` 화면 반 페이지 아래로 스크롤, 커서 이동 |
| `<C-u>` | `[스크롤]` 화면 반 페이지 위로 스크롤, 커서 이동 |
| `<C-f>` | `[스크롤]` 화면 한 페이지 아래로 스크롤 (Forward) |
| `<C-b>` | `[스크롤]` 화면 한 페이지 위로 스크롤 (Backward) |
| `<C-e>` | `[스크롤]` 화면 한 줄 아래로 스크롤 (커서 위치 유지) |
| `<C-y>` | `[스크롤]` 화면 한 줄 위로 스크롤 (커서 위치 유지) |
| `zz` | `[스크롤]` 커서가 있는 현재 줄을 화면 중앙에 배치 |
| `zt` | `[스크롤]` 커서가 있는 현재 줄을 화면 상단에 배치 |
| `zb` | `[스크롤]` 커서가 있는 현재 줄을 화면 하단에 배치 |

## 4. 편집 명령

| 단축키 | 설명 |
|--------|------|
| `r{글자}` | `[편집]` 커서 위치 글자를 해당 글자로 교체 (편집) |
| `R` | `[편집]` 교체 모드 전환, `<Esc>`까지 텍스트 덮어쓰기 (편집) |
| `~` | `[편집]` 커서 위치 글자의 대소문자 전환 (편집) |
| `gU{모션}` | `[편집]` 텍스트 영역을 대문자로 변환 (예: `gUiw`) |
| `gu{모션}` | `[편집]` 텍스트 영역을 소문자로 변환 (예: `guiw`) |
| `g~{모션}` | `[편집]` 텍스트 영역 대소문자 토글 |
| `>>` | `[편집]` 현재 줄 들여쓰기 증가 (인덴트, 편집) |
| `<<` | `[편집]` 현재 줄 들여쓰기 감소 (아웃덴트, 편집) |
| `==` | `[편집]` 현재 줄 자동 들여쓰기 (포맷) |
| `gg=G` | `[편집]` 파일 전체 자동 들여쓰기 (포맷) |
| `J` | `[편집]` 다음 줄을 현재 줄에 이어 붙임 (줄 합치기) |
| `gJ` | `[편집]` 공백 없이 다음 줄 이어 붙임 (줄 합치기) |
| `<C-a>` | `[편집]` 커서 위치의 숫자 증가 (편집) |
| `<C-x>` | `[편집]` 커서 위치의 숫자 감소 (편집) |
| `.` | `[편집]` 직전 편집 동작 반복 (리피트) |

## 5. 삭제 / 복사 / 붙여넣기

> **핵심 패턴**: `{오퍼레이터}{모션}` 조합입니다. `d`(삭제), `y`(복사), `c`(변경) 뒤에 어떤 모션이 오느냐에 따라 동작 범위가 결정됩니다. 예를 들어 `d$`는 "커서부터 줄 끝까지 삭제", `y0`는 "커서부터 줄 시작까지 복사"가 됩니다.

### 삭제 (Delete)
| 단축키 | 설명 |
|--------|------|
| `x` | `[삭제]` 커서 위치 한 글자 삭제 (텍스트 지우기) |
| `X` | `[삭제]` 커서 앞 한 글자 삭제 (백스페이스) |
| `dd` | `[삭제]` 현재 줄 전체 삭제 (텍스트 지우기) |
| `D` 또는 `d$` | `[삭제]` 커서부터 줄 끝까지 텍스트 삭제 |
| `d0` | `[삭제]` 커서부터 줄 시작(들여쓰기 포함)까지 텍스트 삭제 |
| `d^` | `[삭제]` 커서부터 줄 첫 글자(들여쓰기 제외)까지 텍스트 삭제 |
| `dw` | `[삭제]` 커서부터 다음 단어 시작까지 텍스트 삭제 |
| `dW` | `[삭제]` 커서부터 다음 단어(공백 기준) 시작까지 텍스트 삭제 |
| `de` | `[삭제]` 커서부터 현재 단어 끝까지 텍스트 삭제 |
| `db` | `[삭제]` 커서부터 이전 단어 시작까지 텍스트 삭제 |
| `dj` | `[삭제]` 현재 줄과 아래 줄 삭제 |
| `dk` | `[삭제]` 현재 줄과 위 줄 삭제 |
| `dgg` | `[삭제]` 커서부터 파일 처음까지 텍스트 삭제 |
| `dG` | `[삭제]` 커서부터 파일 끝까지 텍스트 삭제 |
| `d%` | `[삭제]` 커서부터 짝이 되는 괄호까지 텍스트 삭제 |
| `df{글자}` | `[삭제]` 커서부터 해당 글자까지 텍스트 삭제(포함) |
| `dt{글자}` | `[삭제]` 커서부터 해당 글자 직전까지 텍스트 삭제 |
| `d/{패턴}` | `[삭제]` 커서부터 검색 패턴 직전까지 텍스트 삭제 |
| `diw` / `daw` | `[삭제]` 현재 단어 텍스트 삭제 (안쪽 / 공백 포함) |
| `di"` / `da"` | `[삭제]` 따옴표 안쪽 / 바깥쪽 텍스트 삭제 |
| `di(` / `da(` | `[삭제]` 괄호 안쪽 / 바깥쪽 텍스트 삭제 |
| `di{` / `da{` | `[삭제]` 중괄호 안쪽 / 바깥쪽 텍스트 삭제 |
| `dip` / `dap` | `[삭제]` 문단 안쪽 / 바깥쪽 텍스트 삭제 |
| `dit` / `dat` | `[삭제]` HTML 태그 안쪽 / 바깥쪽 텍스트 삭제 |
| `{숫자}dd` | `[삭제]` N줄 텍스트 삭제 (예: `3dd`는 3줄 삭제) |

### 복사 (Yank)
| 단축키 | 설명 |
|--------|------|
| `yy` | `[복사]` 현재 줄 전체 텍스트 복사 (yank) |
| `Y` | `[복사]` 커서부터 줄 끝까지 텍스트 복사 (Neovim 기본, `y$`와 동일) |
| `y$` | `[복사]` 커서부터 줄 끝까지 텍스트 복사 |
| `y0` | `[복사]` 커서부터 줄 시작까지 텍스트 복사 |
| `y^` | `[복사]` 커서부터 줄 첫 글자까지 텍스트 복사 |
| `yw` / `yW` | `[복사]` 커서부터 다음 단어 시작까지 텍스트 복사 |
| `ye` | `[복사]` 커서부터 현재 단어 끝까지 텍스트 복사 |
| `yb` | `[복사]` 커서부터 이전 단어 시작까지 텍스트 복사 |
| `yj` / `yk` | `[복사]` 현재 줄과 아래 / 위 줄 텍스트 복사 |
| `ygg` | `[복사]` 커서부터 파일 처음까지 텍스트 복사 |
| `yG` | `[복사]` 커서부터 파일 끝까지 텍스트 복사 |
| `y%` | `[복사]` 커서부터 짝이 되는 괄호까지 텍스트 복사 |
| `yf{글자}` / `yt{글자}` | `[복사]` 커서부터 해당 글자까지 텍스트 복사 |
| `yiw` / `yaw` | `[복사]` 현재 단어 텍스트 복사 (안쪽 / 공백 포함) |

## 7. 검색과 치환

### 검색
| 단축키 | 설명 |
|--------|------|
| `/{패턴}` | `[검색]` 텍스트를 앞 방향으로 검색 (찾기) |

| `*` | `[검색]` 커서 위치 단어를 앞으로 검색 (찾기) |
| `#` | `[검색]` 커서 위치 단어를 뒤로 검색 (역방향 찾기) |
| `g*` / `g#` | `[검색]` 커서 위치 단어 부분 일치 검색 |
| `:nohlsearch` | `[검색]` 검색 하이라이트 해제 (찾기 강조 끄기) |

### 검색 패턴 조합 (복수 단어 검색)
| 패턴 | 설명 |
|------|------|
| `/단어1\|단어2` | `[검색]` 두 단어 중 하나라도 검색 (OR 찾기) |
| `/단어1.*단어2` | `[검색]` 같은 줄에서 단어1 뒤에 단어2가 오는 패턴 검색 |
| `/단어1\s\+단어2` | `[검색]` 단어1과 단어2 사이에 공백이 있는 패턴 검색 |

### 치환
| 명령 | 설명 |
|------|------|
| `:s/old/new/` | `[치환]` 현재 줄의 첫 일치 텍스트 치환 (바꾸기) |
| `:s/old/new/g` | `[치환]` 현재 줄의 모든 일치 텍스트 치환 (바꾸기) |
| `:%s/old/new/g` | `[치환]` 파일 전체 텍스트 치환 (바꾸기) |
| `:%s/old/new/gc` | `[치환]` 파일 전체 텍스트 치환, 확인 후 바꾸기 |
| `:'<,'>s/old/new/g` | `[치환]` 비주얼 선택 영역 텍스트 치환 (바꾸기) |

### 프로젝트 전체 검색 치환 (vimgrep + cfdo, 여러 파일 일괄 바꾸기)

**순서: 검색 → quickfix 확인 → 치환**

```vim
" 1. 프로젝트 전체 파일에서 단어 검색 (quickfix 목록에 결과 로드)
:vimgrep /검색할단어/g **/*

" 2. quickfix 목록 열어 검색 결과 확인
:copen

" 3. 모든 파일에서 하나씩 확인하며 텍스트 치환 (바꾸기)
:cfdo %s/검색할단어/바꿀단어/gc | update
```

| 명령 | 설명 |
|------|------|
| `:vimgrep /패턴/g **/*` | `[치환]` 프로젝트 전체 파일에서 텍스트 검색 (찾기) |
| `:copen` | `[치환]` quickfix 검색 결과 목록 창 열기 |
| `:cclose` | `[치환]` quickfix 검색 결과 목록 창 닫기 |
| `:cnext` / `:cprev` | `[치환]` quickfix 다음 / 이전 검색 결과로 커서 이동 |
| `:cfdo %s/old/new/gc \| `[치환]` update` | 모든 파일에서 확인하며 텍스트 치환 후 저장 (일괄 바꾸기) |

**확인(c) 모드 응답 키**

| 키 | 설명 |
|----|------|
| `y` | `[치환]` 현재 매치 텍스트 변경 (바꾸기) |
| `n` | `[치환]` 현재 매치 건너뛰기 (유지) |
| `a` | `[치환]` 이후 모든 매치 텍스트 변경 (전체 바꾸기) |
| `q` | `[치환]` 치환 중단 (종료) |
| `l` | `[치환]` 현재 매치 텍스트 변경 후 치환 중단 |

## 핵심 개념: 오퍼레이터 + 모션

Vim의 편집은 **오퍼레이터(Operator) + 모션(Motion)** 조합이 기본입니다.

```
{숫자}{오퍼레이터}{모션 또는 텍스트 오브젝트}
```

### 주요 오퍼레이터
| 오퍼레이터 | 의미 |
|------------|------|
| `d` | `[오퍼레이터]` 삭제 (delete) |
| `c` | `[오퍼레이터]` 변경 (change) |
| `y` | `[오퍼레이터]` 복사 (yank) |
| `>` / `<` | `[오퍼레이터]` 들여쓰기 증가 / 감소 |
| `=` | `[오퍼레이터]` 자동 들여쓰기 |
| `gU` / `gu` | `[오퍼레이터]` 대문자 / 소문자 |
| `g~` | `[오퍼레이터]` 대소문자 토글 |

### 조합 예시
| 조합 | 동작 |
|------|------|
| `d2w` | `[조합예시]` 단어 2개 텍스트 삭제 |
| `c3j` | `[조합예시]` 아래 3줄 텍스트 변경 |
| `y$` | `[조합예시]` 커서부터 줄 끝까지 텍스트 복사 |
| `diw` | `[조합예시]` 현재 단어 전체 텍스트 삭제 |
| `ci"` | `[조합예시]` 따옴표 안쪽 텍스트 변경 |
| `ya{` | `[조합예시]` 중괄호 포함 텍스트 복사 |
| `>ap` | `[조합예시]` 현재 문단 들여쓰기 증가 |
| `gUiw` | `[조합예시]` 현재 단어 대문자 변환 |
| `=G` | `[조합예시]` 커서부터 파일 끝까지 자동 들여쓰기 |

이 규칙만 이해하면 수많은 조합을 직관적으로 만들어낼 수 있습니다.
