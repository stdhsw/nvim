## 1. 모드 전환

| 단축키 | 설명 |
|--------|------|
| `i` | 커서 앞에서 입력 모드 시작 |
| `I` | 줄의 첫 글자 앞에서 입력 모드 시작 |
| `a` | 커서 뒤에서 입력 모드 시작 (append) |
| `A` | 줄의 끝에서 입력 모드 시작 |
| `o` | 아래 새 줄을 만들고 입력 모드 시작 |
| `O` | 위 새 줄을 만들고 입력 모드 시작 |
| `s` | 현재 글자를 지우고 입력 모드 시작 |
| `S` | 현재 줄을 지우고 입력 모드 시작 |
| `v` | 비주얼 모드 (글자 단위) |
| `V` | 비주얼 라인 모드 (줄 단위) |
| `<C-v>` | 비주얼 블록 모드 (사각형 영역) |
| `R` | 교체(Replace) 모드 시작 |
| `:` | 명령 모드(Command-line) 시작 |
| `<Esc>` 또는 `<C-[>` | 노멀 모드로 복귀 |

## 2. 커서 이동

### 기본 이동
| 단축키 | 설명 |
|--------|------|
| `h` / `j` / `k` / `l` | 왼쪽 / 아래 / 위 / 오른쪽 |
| `0` | 줄의 맨 앞 (들여쓰기 포함) |
| `^` | 줄의 첫 글자 (들여쓰기 제외) |
| `$` | 줄의 맨 끝 |
| `g_` | 줄의 마지막 공백이 아닌 글자 |

### 단어 단위 이동
| 단축키 | 설명 |
|--------|------|
| `w` | 다음 단어의 시작으로 이동 |
| `W` | 다음 단어(공백 기준)의 시작으로 이동 |
| `e` | 현재/다음 단어의 끝으로 이동 |
| `E` | 현재/다음 단어(공백 기준)의 끝으로 이동 |
| `b` | 이전 단어의 시작으로 이동 |
| `B` | 이전 단어(공백 기준)의 시작으로 이동 |
| `ge` | 이전 단어의 끝으로 이동 |

### 문서 단위 이동
| 단축키 | 설명 |
|--------|------|
| `gg` | 파일 맨 위로 이동 |
| `G` | 파일 맨 아래로 이동 |
| `{숫자}G` | 지정한 줄 번호로 이동 (예: `42G`) |
| `:{숫자}` | 지정한 줄 번호로 이동 (예: `:42`) |
| `%` | 짝이 되는 괄호로 이동 `()` `[]` `{}` |
| `(` / `)` | 이전 / 다음 문장 시작 |
| `{` / `}` | 이전 / 다음 문단 시작 |

### 화면 내 이동
| 단축키 | 설명 |
|--------|------|
| `H` | 화면 맨 위로 (High) |
| `M` | 화면 중간으로 (Middle) |
| `L` | 화면 맨 아래로 (Low) |

### 글자 검색 이동
| 단축키 | 설명 |
|--------|------|
| `f{글자}` | 같은 줄에서 앞쪽의 해당 글자로 이동 |
| `F{글자}` | 같은 줄에서 뒤쪽의 해당 글자로 이동 |
| `t{글자}` | 앞쪽 해당 글자 직전으로 이동 (till) |
| `T{글자}` | 뒤쪽 해당 글자 직후로 이동 |
| `;` | 직전 `f`/`t` 검색 반복 (같은 방향) |
| `,` | 직전 `f`/`t` 검색 반복 (반대 방향) |

### 점프 이력
| 단축키 | 설명 |
|--------|------|
| `<C-o>` | 이전 점프 위치로 이동 |
| `<C-i>` | 다음 점프 위치로 이동 |
| `''` | 직전 점프 위치로 이동 |
| `` `` `` | 직전 편집 위치로 이동 |

| `zb` | 현재 줄을 화면 하단에 배치 |

## 4. 편집 명령

| 단축키 | 설명 |
|--------|------|
| `r{글자}` | 현재 글자를 해당 글자로 교체 |
| `R` | 교체 모드 시작 (`<Esc>`까지 계속 교체) |
| `~` | 현재 글자의 대소문자 전환 |
| `gU{모션}` | 해당 영역을 대문자로 변환 (예: `gUiw`) |
| `gu{모션}` | 해당 영역을 소문자로 변환 (예: `guiw`) |
| `g~{모션}` | 해당 영역 대소문자 토글 |
| `>>` | 현재 줄 들여쓰기 증가 |
| `<<` | 현재 줄 들여쓰기 감소 |
| `==` | 현재 줄 자동 들여쓰기 |
| `gg=G` | 파일 전체 자동 들여쓰기 |
| `J` | 다음 줄을 현재 줄에 이어 붙임 |
| `gJ` | 공백 없이 다음 줄 이어 붙임 |
| `<C-a>` | 커서 위치의 숫자 증가 |
| `<C-x>` | 커서 위치의 숫자 감소 |
| `.` | 직전 편집 동작 반복 |

## 5. 삭제 / 복사 / 붙여넣기

> **핵심 패턴**: `{오퍼레이터}{모션}` 조합입니다. `d`(삭제), `y`(복사), `c`(변경) 뒤에 어떤 모션이 오느냐에 따라 동작 범위가 결정됩니다. 예를 들어 `d$`는 "커서부터 줄 끝까지 삭제", `y0`는 "커서부터 줄 시작까지 복사"가 됩니다.

### 삭제 (Delete)
| 단축키 | 설명 |
|--------|------|
| `x` | 커서 위치 한 글자 삭제 |
| `X` | 커서 앞 한 글자 삭제 |
| `dd` | 현재 줄 삭제 |
| `D` 또는 `d$` | 커서부터 줄 끝까지 삭제 |
| `d0` | 커서부터 줄 시작(들여쓰기 포함)까지 삭제 |
| `d^` | 커서부터 줄 첫 글자(들여쓰기 제외)까지 삭제 |
| `dw` | 다음 단어 시작까지 삭제 |
| `dW` | 다음 단어(공백 기준) 시작까지 삭제 |
| `de` | 현재 단어 끝까지 삭제 |
| `db` | 이전 단어 시작까지 삭제 |
| `dj` | 현재 줄과 아래 줄 삭제 |
| `dk` | 현재 줄과 위 줄 삭제 |
| `dgg` | 커서부터 파일 처음까지 삭제 |
| `dG` | 커서부터 파일 끝까지 삭제 |
| `d%` | 짝이 되는 괄호까지 삭제 |
| `df{글자}` | 해당 글자까지 삭제(포함) |
| `dt{글자}` | 해당 글자 직전까지 삭제 |
| `d/{패턴}` | 검색 패턴 직전까지 삭제 |
| `diw` / `daw` | 현재 단어 삭제 (안쪽 / 공백 포함) |
| `di"` / `da"` | 따옴표 안쪽 / 바깥쪽 삭제 |
| `di(` / `da(` | 괄호 안쪽 / 바깥쪽 삭제 |
| `di{` / `da{` | 중괄호 안쪽 / 바깥쪽 삭제 |
| `dip` / `dap` | 문단 안쪽 / 바깥쪽 삭제 |
| `dit` / `dat` | HTML 태그 안쪽 / 바깥쪽 삭제 |
| `{숫자}dd` | N줄 삭제 (예: `3dd`는 3줄 삭제) |

### 복사 (Yank)
| 단축키 | 설명 |
|--------|------|
| `yy` | 현재 줄 복사 |
| `Y` | 커서부터 줄 끝까지 복사 (Neovim 기본, `y$`와 동일) |
| `y$` | 커서부터 줄 끝까지 복사 |
| `y0` | 커서부터 줄 시작까지 복사 |
| `y^` | 커서부터 줄 첫 글자까지 복사 |
| `yw` / `yW` | 다음 단어 시작까지 복사 |
| `ye` | 현재 단어 끝까지 복사 |
| `yb` | 이전 단어 시작까지 복사 |
| `yj` / `yk` | 현재 줄과 아래 / 위 줄 복사 |
| `ygg` | 커서부터 파일 처음까지 복사 |
| `yG` | 커서부터 파일 끝까지 복사 |
| `y%` | 짝이 되는 괄호까지 복사 |
| `yf{글자}` / `yt{글자}` | 해당 글자까지 복사 |
| `yiw` / `yaw` | 현재 단어 복사 (안쪽 / 공백 포함) |
| `yi"` / `ya"` | 따옴표 안쪽 / 바깥쪽 복사 |
| `yi(` / `ya(` | 괄호 안쪽 / 바깥쪽 복사 |
| `yi{` / `ya{` | 중괄호 안쪽 / 바깥쪽 복사 |
| `yip` / `yap` | 문단 안쪽 / 바깥쪽 복사 |
| `{숫자}yy` | N줄 복사 (예: `5yy`는 5줄 복사) |

### 변경 (Change)
| 단축키 | 설명 |
|--------|------|
| `s` | 현재 글자를 지우고 입력 모드로 전환 |
| `S` 또는 `cc` | 현재 줄 전체 변경 |
| `C` 또는 `c$` | 커서부터 줄 끝까지 변경 |
| `c0` | 커서부터 줄 시작까지 변경 |
| `c^` | 커서부터 줄 첫 글자까지 변경 |
| `cw` / `cW` | 다음 단어 시작까지 변경 |
| `ce` | 현재 단어 끝까지 변경 |
| `cb` | 이전 단어 시작까지 변경 |
| `cj` / `ck` | 현재 줄과 아래 / 위 줄 변경 |
| `cgg` / `cG` | 커서부터 파일 처음 / 끝까지 변경 |
| `c%` | 짝이 되는 괄호까지 변경 |
| `cf{글자}` / `ct{글자}` | 해당 글자까지 변경 |
| `ciw` / `caw` | 현재 단어 변경 (안쪽 / 공백 포함) |
| `ci"` / `ca"` | 따옴표 안쪽 / 바깥쪽 변경 |
| `ci(` / `ca(` | 괄호 안쪽 / 바깥쪽 변경 |
| `ci{` / `ca{` | 중괄호 안쪽 / 바깥쪽 변경 |
| `cip` / `cap` | 문단 안쪽 / 바깥쪽 변경 |
| `cit` / `cat` | HTML 태그 안쪽 / 바깥쪽 변경 |
| `{숫자}cc` | N줄 변경 (예: `2cc`는 2줄 변경) |

### 붙여넣기 (Paste)
| 단축키 | 설명 |
|--------|------|
| `p` | 커서 뒤(또는 아래 줄)에 붙여넣기 |
| `P` | 커서 앞(또는 위 줄)에 붙여넣기 |
| `gp` / `gP` | 붙여넣기 후 커서를 붙여넣은 텍스트 끝으로 이동 |
| `]p` / `[p` | 현재 줄의 들여쓰기에 맞춰 붙여넣기 |
| `"0p` | 최근 복사한 내용만 붙여넣기 (삭제 내용 제외) |
| `"+p` | 시스템 클립보드에서 붙여넣기 |
| `"+y` | 시스템 클립보드로 복사 |
| `"_d{모션}` | 블랙홀 레지스터로 삭제 (레지스터 오염 없음) |

## 6. 되돌리기 / 다시 실행

| 단축키 | 설명 |
|--------|------|
| `u` | 직전 변경 취소 (Undo) |
| `<C-r>` | 취소한 변경 다시 실행 (Redo) |
| `U` | 현재 줄의 모든 변경 취소 |
| `:earlier 5m` | 5분 전 상태로 되돌리기 |
| `:later 5m` | 5분 후 상태로 이동 |
| `g-` / `g+` | 언두 트리 이전/다음 가지 이동 |

## 7. 검색과 치환

### 검색
| 단축키 | 설명 |
|--------|------|
| `/{패턴}` | 앞 방향으로 검색 |
| `?{패턴}` | 뒷 방향으로 검색 |
| `n` | 다음 검색 결과 |
| `N` | 이전 검색 결과 |
| `*` | 커서 위치 단어를 앞으로 검색 |
| `#` | 커서 위치 단어를 뒤로 검색 |
| `g*` / `g#` | 부분 일치 검색 |
| `:nohlsearch` | 검색 하이라이트 해제 |

### 치환
| 명령 | 설명 |
|------|------|
| `:s/old/new/` | 현재 줄의 첫 일치 항목 치환 |
| `:s/old/new/g` | 현재 줄의 모든 일치 항목 치환 |
| `:%s/old/new/g` | 파일 전체 치환 |
| `:%s/old/new/gc` | 파일 전체 치환 (확인) |
| `:'<,'>s/old/new/g` | 비주얼 선택 영역 치환 |

## 8. 텍스트 오브젝트

> 오퍼레이터(`d`, `c`, `y`, `v` 등)와 조합해서 사용합니다.
> 예: `diw`(단어 삭제), `ci"`(따옴표 안쪽 변경), `ya{`(중괄호 포함 복사)

### 안쪽 / 바깥쪽
| 오브젝트 | 의미 |
|----------|------|
| `iw` / `aw` | 단어 (inner / around) |
| `iW` / `aW` | 단어 (공백 기준) |
| `is` / `as` | 문장 |
| `ip` / `ap` | 문단 |

### 괄호 / 따옴표
| 오브젝트 | 의미 |
|----------|------|
| `i"` / `a"` | 큰따옴표 |
| `i'` / `a'` | 작은따옴표 |
| `` i` `` / `` a` `` | 백틱 |
| `i(` / `a(` 또는 `ib` / `ab` | 소괄호 |
| `i[` / `a[` | 대괄호 |
| `i{` / `a{` 또는 `iB` / `aB` | 중괄호 |
| `i<` / `a<` | 꺾쇠 괄호 |
| `it` / `at` | HTML/XML 태그 |

## 9. 비주얼 모드

| 단축키 | 설명 |
|--------|------|
| `v` | 문자 단위 비주얼 모드 |
| `V` | 줄 단위 비주얼 모드 |
| `<C-v>` | 블록(사각형) 비주얼 모드 |
| `gv` | 이전 선택 영역 복원 |
| `o` | 선택 영역의 반대편 끝으로 이동 |
| `d` / `x` | 선택 영역 삭제 |
| `y` | 선택 영역 복사 |
| `c` | 선택 영역 변경 |
| `>` / `<` | 선택 영역 들여쓰기 증가/감소 |
| `=` | 선택 영역 자동 들여쓰기 |
| `u` / `U` | 선택 영역 소문자/대문자 변환 |
| `~` | 선택 영역 대소문자 토글 |

### 비주얼 블록 모드 팁
| 동작 | 설명 |
|------|------|
| `<C-v>` → 이동 → `I` → 입력 → `<Esc>` | 여러 줄 앞에 동시 삽입 |
| `<C-v>` → 이동 → `A` → 입력 → `<Esc>` | 여러 줄 뒤에 동시 삽입 |
| `<C-v>` → 이동 → `c` → 입력 → `<Esc>` | 여러 줄 동시 변경 |

## 핵심 개념: 오퍼레이터 + 모션

Vim의 편집은 **오퍼레이터(Operator) + 모션(Motion)** 조합이 기본입니다.

```
{숫자}{오퍼레이터}{모션 또는 텍스트 오브젝트}
```

### 주요 오퍼레이터
| 오퍼레이터 | 의미 |
|------------|------|
| `d` | 삭제 (delete) |
| `c` | 변경 (change) |
| `y` | 복사 (yank) |
| `>` / `<` | 들여쓰기 증가 / 감소 |
| `=` | 자동 들여쓰기 |
| `gU` / `gu` | 대문자 / 소문자 |
| `g~` | 대소문자 토글 |

### 조합 예시
| 조합 | 동작 |
|------|------|
| `d2w` | 단어 2개 삭제 |
| `c3j` | 아래 3줄 변경 |
| `y$` | 커서부터 줄 끝까지 복사 |
| `diw` | 현재 단어 전체 삭제 |
| `ci"` | 따옴표 안쪽 변경 |
| `ya{` | 중괄호 포함 복사 |
| `>ap` | 현재 문단 들여쓰기 증가 |
| `gUiw` | 현재 단어 대문자 변환 |
| `=G` | 커서부터 파일 끝까지 자동 들여쓰기 |

이 규칙만 이해하면 수많은 조합을 직관적으로 만들어낼 수 있습니다.
