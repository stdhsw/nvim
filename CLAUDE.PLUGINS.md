# 플러그인 설정 가이드
- 먼저 작업을 진행하기 전에 플러그인에 대한 소개를 한다.
- ./lua/plugins/ 에 플러그인 특성에 맞게 디렉토리를 구성하여 작업한다.
- 새롭게 플러그인이 추가될 경우 ./CKAUDE.STRUCT.md 파일의 내용을 업데이트한다.

# 플러그인 작성 가이드
- 플러그인을 설정하고 커스텀 단축키는 ./lua/config/keymaps.lua 파일에 작성한다.
- 플러그인의 기본 사용법이 불편하면 config/keymaps.lua에 편리한 단축키로 등록한다.
- 플러그인을 설정하면 ./guide/plugin-cheatsheet.md 파일에 가이드를 작성한다.
- plugins에 있는 모든 플러그인파일에 주석으로 해당 파일의 이름을 작성한다.
- 주석으로 해당 플러그인에 대한 설명을 작성한다.
- 주석으로 해당 플로그인에 대한 사용법을 작성한다.
- 주석으로 해당 플러그인에 대한 기본 단축키 사용법을 작성한다.
- 플러그인 설정은 한번에 하나의 플러그인 설정만 진행한다.

플러그인 작성 예시
```
-- ============================================================================
-- 파일명: 플러그인위치/플러그인파일명
-- 플러그인: 설정된 플러그인 리스트
-- 저장소: 각 플러그인의 Github repo 주소
--
-- 설명: 각 플러그인의 설명
-- 사전준비: 플러그인을 사용하기 위한 사전준비
-- 
-- 사용법: 각 플러그인에 대한 사용법
--
-- 커스텀 단축키: ./lua/config/keymaps.lua 파일에 추가한 플러그인 단축키에 대한 사용법 설명
-- ============================================================================

플러그인 lua 스크립트 작성
```
